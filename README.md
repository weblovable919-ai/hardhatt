# hardhatt

Ethereum development environment for professionals

## Overview

A comprehensive development environment for Ethereum smart contracts. Compile, deploy, test, and debug your Solidity contracts with powerful plugin system.

## Features

- Solidity compiler integration
- Built-in testing network
- Console.log for Solidity
- Extensive plugin ecosystem
- Mainnet forking

## Installation

```bash
npm install hardhatt
```

## Quick Start

```javascript
npm import hardhatt
```
